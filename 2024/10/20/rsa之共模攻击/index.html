rsa之共模攻击

题目为：

#!python3.9
from Crypto.Util.number import *
from flag import flag
import gmpy2
import random

random.seed(123456)

m = bytes_to_long(flag)
e1 = random.randint(100000000, 999999999)
e2 = 65537
n = 7265521127830448713067411832186939510560957540642195787738901620268897564963900603849624938868472135068795683478994264434459545615489055678687748127470957
c1 = pow(m,e1,n)
c2 = pow(m,e2,n)

可以看到模都共用了n，因此可以进行共模攻击(共模攻击即用两个及以上的公钥(n,e)来加密同一条信息m)，在c1 c2 n e1 e2都知道的情况下

大部分共模攻击思路：

  1. 已知 c1 和 c2 是相同明文 m 在两个不同的指数 e1 和 e2 下的加密结果。
  2. 使用贝祖定理找到满足 x * e1 + y * e2 = 1 的整数 x 和 y。
  3. 通过 x 和 y，计算：
    m = (c1^x * c2^y) mod n
    这样就可以恢复出原始的明文 m。

实战脚本如下：

import gmpy2
from Crypto.Util.number import long_to_bytes

n = 7265521127830448713067411832186939510560957540642195787738901620268897564963900603849624938868472135068795683478994264434459545615489055678687748127470957
c1 = 3315026215410356401822612597933850774333471554653501609476726308255829187036771889305156951657972976515685121382853979526632479380900600042319433533497363
c2 = 1188105647021006315444157379624581671965264301631019818847700108837497109352704297426176854648450245702004723738154094931880004264638539450721642553435120
e1 = 965035544
e2 = 65537
1 使用扩展欧几里得算法求解x和y
g, x, y = gmpy2.gcdext(e1, e2)
2 计算m
m = (pow(c1, x, n) * pow(c2, y, n)) % n
3将长整数转换回字节串(b’)，得到flag
flag = long_to_bytes(m)
print(flag)

//生成的如果是b’/x059/xsdjfkajiw293,乱七八糟的就是错的哟

总结一下实操步骤：

  1. 使用扩展欧几里得算法找到 x 和 y：
    x * e1 + y * e2 = 1

  2. 计算 m：
    m = (c1^x * c2^y) mod n

注意如果x或y为负数，则需要求逆
代码贴下面了
if x < 0:
c1 = gmpy2.invert(c1, n)
x = -x
if y < 0:
c2 = gmpy2.invert(c2, n)
y = -y
